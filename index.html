📖 좌석 배치표 사용 설명서

×
  • 📋 명단 입력 및 엑셀 붙여넣기: 엑셀에서 학생 명단(예: 1 홍길동)을 복사하여 텍스트 상자에 붙여넣은 뒤 [적용]을 누르세요. 혹은 화면의 각 좌석에 직접 번호와 이름을 입력할 수도 있습니다.
  • 🎨 환경 설정 (테마, 교실 구조): 상단에서 디자인 테마나 열별 색상 구분을 설정합니다. 학급에 맞게 행(줄)과 열(분단)의 수를 조절할 수 있으며, 실제 구조와 같이 '문 위치(교사시점)'도 지정 가능합니다.
  • 🖱️ 자리 이동 (드래그 앤 드롭): 학생 화면(첫 번째 페이지)의 좌석에 배치된 학생의 자리를 마우스로 클릭해 끌어서 다른 학생의 자리나 빈 자리 위에 떨어뜨리면 즉시 두 자리가 맞바뀝니다.
  • 🔒 특정 자리 고정 (자물쇠): 섞기 기능을 사용할 때 특정 학생의 자리를 변경하지 않고 고정하려면, 각 책상 우측 상단의 자물쇠 아이콘(🔓)을 눌러 잠금 상태(🔒)로 만드세요.
  • ✨ 특수 배치: [✨ 특수 배치] 버튼을 눌러 특정 번호의 학생들을 지정된 구역(예: 맨 앞줄, 맨 뒷줄 등)에 먼저 배치하는 사전 규칙을 만들 수 있습니다.
  • 🔀 무작위 섞기: 자물쇠가 잠긴 자리와 특수 배치 대상을 먼저 배정한 뒤, 나머지 빈자리와 학생들을 완벽히 랜덤하게 뒤섞어 새롭게 배정합니다.
  • 💾 진행상태 저장 및 불러오기: 작업 중인 배치표를 파일로 [💾 저장]해 두면, 추후 [📂 불러오기]를 통해 언제든 이어서 수정하고 관리할 수 있습니다.
  • 🖨️ 인쇄: [🖨️ 일반 인쇄] 버튼을 누르면 화면에 보이는 학생용과 교사용(거울 반전) 배치표를 바로 출력할 수 있습니다.
  • 🔐 비밀 배치 규칙 (Ctrl+Shift+K): 키보드에서 Ctrl+Shift+K를 누르면 비밀 관리자 패널이 열립니다. 여기서 학생들 모르게 특정 학생을 서로 2칸 이상 떨어뜨리거나, 지정 줄·열에 배정하는 규칙을 설정할 수 있습니다. 설정값은 이 브라우저에 자동 저장되며, 섞기 시 가장 먼저 적용됩니다.
